nv_zim

User guide

This page refers to the latest nv_zim release. You can open it with Help > Zim connection Online help.

nv_zim is a plugin that connects novelibre projects with a Zim Desktop Wiki. This is mainly intended for world building documentation.

The plugin adds an Zim Desktop Wiki entry to the novelibre Tools menu, and a Zim connection Online help entry to the Help menu. The property views of characters, locations, items, and book get a Wiki page button. The Toolbar gets a button.
