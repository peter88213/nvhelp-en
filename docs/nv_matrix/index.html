nv_matrix

User guide

This page refers to the latest nv_matrix release. You can open it with Help > Matrix plugin Online help.

The plugin adds a Matrix entry to the novelibre Tools menu, and a Matrix plugin Online help entry to the Help menu. The Toolbar gets a button.

Start the matrix manager

  • Open the matrix manager either from the main menu: Tools > Matrix,

  • or via the button in the toolbar.

Add/remove relationships

  • Add/Remove relationships by klicking on the nodes with the Ctrl key pressed.

Mouse wheel scrolling

  • Use the mouse wheel for vertical scrolling.

  • Use the mouse wheel with the Shift key pressed for horizontal scrolling.

The footer bar

In the footer bar you will find several checkboxes with which you can deselect categories for display, as well as a button to close the window. The states of the checkboxes are retained after closing.

Exit

  • Close the window.
