Help menu
=========

**Quick access to the help pages for novelibre and its plugins**

.. figure:: _images/help_menu01.png
   :alt: novelibre screenshot

.. note:: 
   The *Help* menu can be extended by plugins adding links to their
   specific help pages.

Online Help
-----------

With **Help > Online Help**,
you can launch your system web browser with the *novelibre* online help URL.

novelibre home page
-------------------

With **Help > novelibre home page**
you can launch your system web browser with the *novelibre* home page URL.

.. hint::
   This is a quick way to get to the download link of the most recent 
   *novelibre* version.
